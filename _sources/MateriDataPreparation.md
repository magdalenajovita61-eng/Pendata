# Materi Data Preparation

Tahap **Data Preparation** berfokus pada menyiapkan data agar siap dipakai untuk modeling.

## 1. Select Data
Memilih data yang relevan dan membuang yang tidak diperlukan.

## 2. Clean Data
Membersihkan data dari error.

**Contoh:**
- Mengisi atau menghapus missing value  
- Menghapus duplikat  
- Memperbaiki format salah  

## 3. Construct Data
Membuat fitur baru (feature engineering).

**Contoh:**
- Menghitung umur dari tanggal lahir  
- Menggabungkan beberapa kolom  

## 4. Integrate Data
Menggabungkan data dari berbagai sumber.

**Contoh:**
- Join tabel  
- Merge dataset  

## 5. Format Data
Mengubah format data agar sesuai kebutuhan model.

**Contoh:**
- Normalisasi  
- Encoding kategorikal  
- Dataset description
