# Materi Modeling

Tahap **Modeling** adalah proses membangun model machine learning.

## 1. Select Modeling Techniques
Menentukan algoritma yang akan digunakan.

**Contoh:**
- Regresi  
- KNN  
- Decision Tree  
- Random Forest  

## 2. Generate Test Design
Menentukan metode evaluasi.

**Contoh:**
- Train-test split  
- Cross-validation  

## 3. Build Model
Melatih model dengan data training.

**Output:**
- Model terlatih  
- Parameter awal  

## 4. Assess Model
Menilai performa model.

**Contoh metrik:**
- Akurasi  
- Precision  
- Recall  
- RMSE  

Jika performa buruk, lakukan tuning parameter.
