# Materi Deployment

Tahap **Deployment** adalah penerapan model ke lingkungan nyata.

## 1. Plan Deployment
Merencanakan bagaimana model digunakan.

**Contoh:**
- Website  
- API  
- Aplikasi internal  

## 2. Plan Monitoring and Maintenance
Menyiapkan rencana monitoring.

**Contoh:**
- Mengecek performa model secara berkala  
- Update model jika performa menurun  

## 3. Produce Final Report
Membuat laporan akhir proyek.

**Isi laporan:**
- Tujuan  
- Metode  
- Hasil  
- Kesimpulan  

## 4. Review Project Experience
Mereview pengalaman proyek.

**Tujuan:**
- Menjadi pembelajaran untuk proyek berikutnya  
- Dokumentasi pengalaman
