Proyek Missing Values Inputation + Materi Normalisasi data#

Missing Value Imputation#

a. Missing Value Imputation#

Missing Value Imputation adalah proses mengisi atau mengganti nilai data yang hilang dalam suatu dataset agar data tersebut tetap dapat digunakan dalam proses analisis. Nilai yang hilang biasanya terjadi karena data tidak tercatat, respon yang tidak lengkap, atau kesalahan saat pengumpulan data. Jika nilai yang hilang tidak ditangani, hal ini dapat mempengaruhi hasil analisis dan mengurangi kualitas data. Oleh karena itu, dilakukan proses imputasi untuk memperkirakan nilai yang hilang berdasarkan data yang tersedia. Pada analisis ini, proses penanganan missing value dilakukan menggunakan metode Weighted K-Nearest Neighbor (WKNN), yaitu metode yang mengisi nilai yang hilang dengan mempertimbangkan kedekatan dan bobot dari data tetangga terdekat sehingga nilai yang dihasilkan lebih representatif.

b. Metode Weighted K-Nearest Neighbor (WKNN)#

Weighted K-Nearest Neighbor (WKNN) adalah pengembangan dari metode K-Nearest Neighbor (KNN) yang digunakan untuk memperkirakan nilai yang hilang dengan memberikan bobot pada setiap data tetangga terdekat. Pada metode ini, data yang memiliki jarak lebih dekat dengan data yang memiliki missing value akan memberikan kontribusi yang lebih besar dibandingkan data yang jaraknya lebih jauh.

Dengan adanya pembobotan tersebut, proses estimasi nilai menjadi lebih representatif karena mempertimbangkan tingkat kedekatan antar data. Semakin dekat jarak suatu data dengan data yang memiliki nilai hilang, maka semakin besar pengaruhnya dalam menentukan nilai pengganti.

Secara umum, langkah-langkah dalam metode WKNN adalah sebagai berikut:

  1. Menghitung jarak antara data yang memiliki missing value dengan data lain dalam dataset.

  2. Mengurutkan data berdasarkan nilai jarak dari yang paling kecil.

  3. Menentukan bobot untuk setiap tetangga berdasarkan jaraknya.

  4. Menghitung nilai imputasi menggunakan rata-rata berbobot dari tetangga terdekat.

c. Hitungan Menggunakan Microsoft Excel#

Pada analisis ini, proses imputasi missing value juga dilakukan secara manual dengan bantuan Microsoft Excel untuk mempermudah perhitungan. Excel digunakan untuk menghitung jarak antar data, menentukan bobot, serta menghitung nilai imputasi berdasarkan metode Weighted K-Nearest Neighbor (WKNN). Adapun tahapan perhitungannya adalah sebagai berikut:

  1. Menormalisasikan data sebelum menghitung jarak.

  2. Menghitung jarak antara data yang memiliki nilai kosong dengan data lainnya menggunakan rumus Euclidean Distance.

  3. Mengurutkan hasil perhitungan jarak dari yang paling kecil untuk mengetahui tetangga terdekat.

  4. Menghitung bobot masing-masing tetangga menggunakan rumus bobot W = 1/d, yang dimana d adalah jarak antar data.

  5. Mengalikan bobot dengan nilai atribut dari setiap data tetangga.

  6. Menjumlahkan seluruh hasil perkalian bobot dengan atribut serta total bobot.

  7. Menentukan nilai imputasi dengan menggunakan rata-rata berbobot dari tetangga terdekat.

Hasil perhitungan manual pada 10 data diabetes yang sudah di normalisasi menggunakan Microsoft Excel dapat dilihat pada tabel perhitungan yang saya buat pada gambar berikut.

d. Perhitungan Menggunakan Python#

Pada tahap ini, proses imputasi missing value dilakukan menggunakan Python untuk mempermudah perhitungan metode Weighted K-Nearest Neighbor (WKNN) secara otomatis. Program dimulai dengan mengimpor beberapa library yang dibutuhkan, yaitu Pandas untuk membaca dan mengolah data, NumPy untuk menangani nilai numerik dan missing value, serta math untuk perhitungan akar dan jarak Euclidean. Setelah itu, dataset diabetes dibaca dari file diabetes.csv, kemudian diambil 10 data pertama agar sesuai dengan data yang digunakan pada perhitungan manual di Microsoft Excel.

Dataset yang digunakan terdiri dari beberapa variabel, yaitu Pregnancies, Glucose, BMI, Age, dan Outcome. Variabel Pregnancies, Glucose, BMI, dan Age digunakan sebagai atribut fitur, sedangkan Outcome digunakan sebagai target atau kelas. Selanjutnya, data pada kolom fitur dinormalisasi menggunakan metode Min-Max agar seluruh nilai berada pada rentang 0 sampai 1, sehingga perbedaan skala antar atribut tidak memengaruhi proses perhitungan jarak. Sementara itu, kolom Outcome tidak dinormalisasi karena berupa data kelas.

Setelah proses normalisasi, program membuat dua missing value secara sengaja, yaitu pada BMI data ke-8 dan Outcome data ke-4. Kedua nilai yang hilang tersebut kemudian akan diisi menggunakan metode WKNN. Selanjutnya, nilai K ditentukan dengan rumus akar dari jumlah data (√n), sehingga dari 10 data diperoleh nilai K = 3. Nilai ini digunakan untuk menentukan jumlah tetangga terdekat yang akan dipakai dalam proses imputasi.

Berikutnya, program menghitung jarak Euclidean antara data yang memiliki missing value dengan data lainnya. Untuk imputasi BMI pada data ke-8, perhitungan jarak dilakukan menggunakan atribut Pregnancies, Glucose, Age, dan Outcome, karena kolom BMI sedang kosong. Setelah jarak diperoleh, dihitung bobot masing-masing tetangga dengan rumus w = 1/d, di mana data yang memiliki jarak lebih dekat akan memperoleh bobot lebih besar. Nilai imputasi BMI kemudian dihitung menggunakan rata-rata berbobot dari K tetangga terdekat.

Sementara itu, untuk imputasi Outcome pada data ke-4, perhitungan jarak dilakukan menggunakan atribut Pregnancies, Glucose, BMI, dan Age. Dalam proses ini, data ke-8 tidak digunakan karena nilai BMI pada data tersebut masih kosong. Setelah diperoleh K tetangga terdekat, bobot dari masing-masing kelas dijumlahkan. Kelas dengan jumlah bobot terbesar dipilih sebagai hasil imputasi Outcome. Terakhir, hasil imputasi untuk BMI data ke-8 dan Outcome data ke-4 dimasukkan kembali ke dalam dataset, sehingga diperoleh data yang sudah lengkap tanpa missing value.

import pandas as pd
import numpy as np
from math import sqrt

# =========================================================
# 1. LOAD DATA
# =========================================================
df = pd.read_csv("diabetes.csv")

# Ambil 10 data yang sama seperti manual Excel
data = df.head(10).copy().reset_index(drop=True)

# Kolom yang dipakai
feature_cols = ["Pregnancies", "Glucose", "BMI", "Age"]
target_col = "Outcome"

# =========================================================
# 2. NORMALISASI MIN-MAX
# =========================================================
norm = data[feature_cols].astype(float).copy()
norm = (norm - norm.min()) / (norm.max() - norm.min())

# Target tidak dinormalisasi
norm[target_col] = data[target_col].astype(int)

# Tambahkan id agar sama seperti Excel
norm.insert(0, "id", range(1, len(norm) + 1))

# =========================================================
# 3. BUAT MISSING VALUE
# =========================================================
# Missing fitur: BMI data ke-8
norm.loc[7, "BMI"] = np.nan

# Missing target: Outcome data ke-4
norm.loc[3, "Outcome"] = np.nan

print("=== DATA NORMALISASI DENGAN MISSING VALUE ===")
print(norm)
print()

# =========================================================
# 4. NILAI K
# =========================================================
n = len(norm)
k_real = sqrt(n)
k = round(k_real)

print("=== NILAI K ===")
print(f"K = sqrt({n}) = {k_real:.8f} -> {k}")
print()

# =========================================================
# 5. FUNGSI JARAK EUCLIDEAN
# =========================================================
def euclidean_distance(row1, row2, cols):
    total = 0.0
    for col in cols:
        total += (row1[col] - row2[col]) ** 2
    return sqrt(total)

# =========================================================
# 6. WKNN UNTUK MISSING BMI DATA KE-8
# =========================================================
target_bmi_idx = 7   # data ke-8

target_bmi_row = norm.loc[target_bmi_idx]

bmi_distance_cols = ["Pregnancies", "Glucose", "Age", "Outcome"]

bmi_rows = []
for i in range(len(norm)):
    if i == target_bmi_idx:
        continue  # skip dirinya sendiri

    row = norm.loc[i]

    dist = euclidean_distance(row, target_bmi_row, bmi_distance_cols)
    weight = 1 / dist if dist != 0 else 0
    nilai_imputasi = weight * row["BMI"]

    bmi_rows.append({
        "data": f"s{i+1}",
        "jarak": dist,
        "bobot": weight,
        "nilai_imputasi": nilai_imputasi
    })

bmi_table = pd.DataFrame(bmi_rows)
bmi_table["ranking"] = bmi_table["jarak"].rank(method="min", ascending=True).astype("Int64")
bmi_table = bmi_table.sort_values("data", key=lambda s: s.str.extract(r"(\d+)").astype(int)[0]).reset_index(drop=True)

print("=== TABEL MANUAL WKNN UNTUK BMI DATA KE-8 ===")
print(bmi_table.to_string(index=False))
print()

# Ambil top K berdasarkan jarak terkecil
bmi_topk = bmi_table.nsmallest(k, "jarak").copy()

sum_bobot_bmi = bmi_topk["bobot"].sum()
sum_nilai_bmi = bmi_topk["nilai_imputasi"].sum()
imputasi_bmi = sum_nilai_bmi / sum_bobot_bmi

print("=== TOP K BMI ===")
print(bmi_topk.to_string(index=False))
print()
print(f"Jumlah bobot top K       = {sum_bobot_bmi:.6f}")
print(f"Jumlah nilai imputasi    = {sum_nilai_bmi:.6f}")
print(f"Hasil imputasi BMI s8    = {imputasi_bmi:.6f}")
print()

# =========================================================
# 7. WKNN UNTUK MISSING TARGET OUTCOME DATA KE-4
# =========================================================
target_outcome_idx = 3   # data ke-4

target_outcome_row = norm.loc[target_outcome_idx]

outcome_distance_cols = ["Pregnancies", "Glucose", "BMI", "Age"]

outcome_rows = []
for i in range(len(norm)):
    if i == target_outcome_idx:
        continue  # skip dirinya sendiri
    if i == 7:
        continue  # s8 dikeluarkan karena BMI missing

    row = norm.loc[i]

    dist = euclidean_distance(row, target_outcome_row, outcome_distance_cols)
    weight = 1 / dist if dist != 0 else 0
    kelas_x_bobot = weight * row["Outcome"]

    outcome_rows.append({
        "data": f"s{i+1}",
        "jarak": dist,
        "bobot": weight,
        "kelas_x_bobot": kelas_x_bobot,
        "kelas": int(row["Outcome"])
    })

outcome_table = pd.DataFrame(outcome_rows)
outcome_table["ranking"] = outcome_table["jarak"].rank(method="min", ascending=True).astype(int)
outcome_table = outcome_table.sort_values("data", key=lambda s: s.str.extract(r"(\d+)").astype(int)[0]).reset_index(drop=True)

print("=== TABEL MANUAL WKNN UNTUK OUTCOME DATA KE-4 ===")
print(outcome_table.to_string(index=False))
print()

# Ambil top K
outcome_topk = outcome_table.nsmallest(k, "jarak").copy()

# Jumlah bobot per kelas
bobot_kelas_0 = outcome_topk.loc[outcome_topk["kelas"] == 0, "bobot"].sum()
bobot_kelas_1 = outcome_topk.loc[outcome_topk["kelas"] == 1, "bobot"].sum()

imputasi_outcome = 1 if bobot_kelas_1 > bobot_kelas_0 else 0

print("=== TOP K OUTCOME ===")
print(outcome_topk.to_string(index=False))
print()
print(f"Jumlah bobot kelas 0 = {bobot_kelas_0:.6f}")
print(f"Jumlah bobot kelas 1 = {bobot_kelas_1:.6f}")
print(f"Hasil imputasi Outcome s4 = {imputasi_outcome}")
print()

# =========================================================
# 8. MASUKKAN HASIL IMPUTASI KE DATA
# =========================================================
hasil = norm.copy()
hasil.loc[7, "BMI"] = imputasi_bmi
hasil.loc[3, "Outcome"] = imputasi_outcome

print("=== DATA SETELAH IMPUTASI ===")
print(hasil)

2. Normalisasi Data#

Pengertian Normalisasi Data#

Normalisasi data adalah proses mengubah skala nilai data menjadi rentang tertentu tanpa mengubah informasi utama dari data tersebut.

Normalisasi biasanya digunakan dalam machine learning untuk:

  • menyamakan skala antar fitur

  • mencegah fitur dengan nilai besar mendominasi perhitungan

  • meningkatkan performa algoritma berbasis jarak seperti KNN / WKNN

Contoh masalah jika tidak dinormalisasi:

Macam Macam Normalisasi#

1. Min-Max Normalization#

Min-Max Normalization adalah metode normalisasi yang mengubah nilai data ke dalam rentang tertentu (biasanya 0 sampai 1) dengan menggunakan nilai minimum dan maksimum dari data.

Metode ini mempertahankan proporsi jarak antar data sehingga hubungan antar data tetap sama.

2. Z-Score Normalization (Standardization)#

Z-score normalization adalah metode normalisasi yang mengubah nilai data berdasarkan rata-rata (mean) dan standar deviasi.

Metode ini menghasilkan data dengan:

  • mean = 0

  • standar deviasi = 1

Metode ini cocok digunakan jika data memiliki distribusi normal.
